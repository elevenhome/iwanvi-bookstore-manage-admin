# 中文万维图书资源中心管理系统

### 技术选型
基于Spring Cloud的微服务架构:

* 核心框架：Spring Boot，Spring Cloud
* 安全框架：Apache Shiro
* 模板引擎：Thymeleaf
* 持久层框架：MyBatis
* 数据库和连接池：MySQL，Druid
* 定时任务:Quartz
* 工具类：Fastjson
* 分页插件：PageHelper
* 接口管理：Swagger2
* 验证码插件：Kaptcha
